## Quick Start

All commands are run from the root of the project, from a terminal:

| Command                | Action                                           |
| :--------------------- | :----------------------------------------------- |
| `npm install`          | Installs dependencies                            |
| `npm run dev`          | Run the development server                       |
| `npm run vercel`       | Run in the same conditions as Vercel             |
